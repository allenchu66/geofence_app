隱私權政策

本應用程式「共享家人位置」致力於保護您的個人隱私與資料安全。以下說明我們如何收集、使用、儲存及保護您的資料，請您詳細閱讀。

1. 資料收集方式與項目

本應用於運作過程中會收集以下資訊：

  • 帳號資料：使用 Firebase Authentication 收集並儲存 Email、暱稱與大頭貼資訊，用於登入與識別身份。
  • 即時位置：在您授權後，應用會持續追蹤您的 GPS 位置，用於即時位置共享與地理圍欄通知。
  • 位置歷史：會儲存每日的定位紀錄，供您日後查閱。
  • Geofence 設定：您對好友所設定的地理圍欄（名稱、半徑、經緯度）會儲存於 Firebase Firestore。
  • 推播裝置識別碼（FCM Token）：用於接收通知訊息。
  • 錯誤紀錄與偵錯資訊：在 App 發生錯誤時，我們可能會收集設備型號、Android 版本與例外錯誤訊息以改善服務。

2. 資料使用目的

  • 提供即時位置共享功能。
  • 在對象進入／離開地理圍欄時傳送通知。
  • 顯示歷史定位紀錄與區域聚集分析。
  • 提升系統穩定性與錯誤排除效率。
  • 不會用於廣告用途，亦不會將您的資料販售給第三方。

3. 權限使用說明

  • 定位權限（精確與背景定位）：用於追蹤您的即時位置與觸發地理圍欄事件。僅於您授權後使用。
  • 通知權限：用於提示您或對方進入／離開地理圍欄的狀況。
  • 網路權限：用於與 Firebase 同步資料，包括帳號、位置、Geofence 與通知設定。

4. 資料儲存與安全措施

  • 所有資料皆儲存在 Google Firebase Cloud Firestore 與 Firebase Storage，並受其安全機制保護。
  • 所有通訊均透過加密傳輸（HTTPS/TLS）進行。
  • 僅限經授權之使用者可存取個人資料。
  • 您可以隨時透過 App 介面刪除帳號與所有相關資料。

5. 使用者權利

  • 您有權查閱、下載、修改或刪除您所提供的個人資料。
  • 您可隨時終止位置共享、移除 Geofence 設定，或撤銷授權權限。
  • 您可聯繫我們要求刪除所有紀錄與帳號資訊。

6. 第三方服務說明

本 App 僅使用下列第三方服務：

  • Firebase Authentication：帳號驗證
  • Firebase Cloud Firestore：儲存位置、使用者資料與設定
  • Firebase Cloud Messaging (FCM)：推播通知
  • Firebase Storage：大頭貼儲存

7. 政策變更

我們保留隨時更新本隱私政策之權利。您可隨時於此頁查看最新版本。
